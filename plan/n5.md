# N5 — Right-half experience path

NODE N5 deps=N0 done=pr-experience.sh lane=agent :: Right-half experience path — commission PRs must carry "how to see it working" (G5)

## Intent

The *right* half of fidelity, made routine on every commission PR: a PR must carry **"how to see
it working"** — the concrete path for the Operator to experience the change, not just read that
tests pass. Grounding *right* (matches what the Operator meant) is the Operator's judgment; this
node ensures every PR hands the Operator the means to make that judgment cheaply (fewer turns to
convey acceptance).
