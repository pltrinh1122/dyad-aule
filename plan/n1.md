# N1 — Frontier deriver

NODE N1 deps=N0 done=frontier.sh lane=agent :: Frontier deriver — mechanize node-state derivation (criteria-on-main; dispositions) and surface the ready frontier as a d-start seam (surfaced, never auto-executed)

## Intent

Mechanize the queue's contract: a tool (`bin/_frontier`) that derives each node's state from
ground every time it runs — `done=<file>.sh` DONE ⟺ that criteria file is tracked on
`origin/main`; `done=disposition` DONE ⟺ the DISPOSITION line is non-TODO; READY ⟺ every dep is
DONE. The **frontier** (the READY set) is surfaced as a `d-start` seam — printed, never
auto-executed (`auto-trigger ≠ auto-judgment`; picking a node stays a disposition).

deps=N0 because the deriver needs the plan format and kind-home to exist first.
