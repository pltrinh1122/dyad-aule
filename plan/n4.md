# N4 — dyad-rt bootstrap (contingent on N2)

NODE N4 deps=N2 done=rt-bootstrap.sh lane=agent :: dyad-rt bootstrap — propagate floor/wrappers/enforcer/check/CI into a fresh commission repo (contingent: executes only if N2 disposes commission-per-repo; else struck by replan)

## Intent

Propagate the dyad-rt runtime (the `.githooks` floor, `bin/` wrappers, the `bin/_dyad-rt`
enforcer, `./check`, and the CI workflow) into a fresh commission repo, so a new commission is
born already guarded. **Contingent:** this node executes only if N2 disposes *commission-per-repo*;
if N2 chooses the in-repo product kind, N4 is struck by replan. deps=N2 makes that contingency
structural — N4 cannot be READY until the Operator disposes N2.
