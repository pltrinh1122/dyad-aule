# N6 — Telos metering

NODE N6 deps=N0 done=telos-metering.sh lane=agent :: Telos metering — reflect/ ledger convention: turns-to-spine / turns-to-criteria / rework, per node (G7)

## Intent

Make the telos measurable: a `reflect/` ledger convention that records, per node, **turns-to-spine**
(how many operator↔agent turns to pin the intent), **turns-to-criteria** (turns to stand up the
executable acceptance check), and **rework** (turns spent re-doing). The craft drives turns toward
1; without metering, "fewer turns" is a slogan. This node makes the ledger's home
(`reflect/`, SPAOR Reflect) carry the numbers the telos claims to minimize.
